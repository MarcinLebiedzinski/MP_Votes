Polish Parliament

The application presents basic information about the Polish Parliament based on the API of the government website.

Application let us to check some basic informations:

- terms list, clubs list, member's details

- voting list and mebers's votes

- member's education report, member's age report, absences report

The data was limited to the 9th and 10th terms due to the amount of information provided by the API
